<!DOCTYPE html> and <html>

<!DOCTYPE html> tells browser this is HTML5.

<html> is the root of the entire webpage.

<head>

Contains metadata like title, SEO info, links, scripts.

<title>

Defines the browser tab name.

<body>

All visible content goes inside body.

<header>

Top section of a page (logo, heading, navigation).

<nav>

Contains navigation links.

Home | About

<main>

Main content of the page (only one per page recommended).

<section>

Groups related content into sections.

<article>

Independent content like blog post or news article.

Example Article

This is an article content.

<aside>

Side content like sidebar or ads.

Sidebar info here

<footer>

Bottom section of page (copyright, links).
